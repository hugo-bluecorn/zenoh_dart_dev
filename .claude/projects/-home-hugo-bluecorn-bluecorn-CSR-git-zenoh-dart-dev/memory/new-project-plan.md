## Execution Steps

### 1. Create monorepo root + git init
```bash
mkdir -p /home/hugo-bluecorn/bluecorn/CSR/git/zenoh-dart
cd /home/hugo-bluecorn/bluecorn/CSR/git/zenoh-dart
git init
```

### 2. Set up FVM
```bash
fvm use stable
```

### 3. Create root pubspec.yaml
```yaml
name: _zenoh_workspace
publish_to: none
environment:
  sdk: ^3.11.0
workspace:
  - packages/*
dev_dependencies:
  melos: ^7.4.0
```
Then: `fvm dart pub get`

### 4. Create zenoh package
```bash
cd packages && fvm dart create --template=package zenoh && cd ..
```

Customize packages/zenoh/pubspec.yaml:
```yaml
name: zenoh
description: "Dart bindings for the Zenoh pub/sub/query protocol via FFI."
version: 0.0.1
repository: https://github.com/hugo-bluecorn/zenoh-dart
resolution: workspace
environment:
  sdk: ^3.11.0
dependencies:
  args: ^2.6.0
  ffi: ^2.1.3
dev_dependencies:
  ffigen: ^20.1.1
  test: ^1.25.0
  lints: ^5.1.0
```

### 5. Melos bootstrap
```bash
fvm dart run melos bootstrap
```

### 6. Add 5 submodules
- extern/zenoh-c @ v1.7.2
- extern/zenoh-cpp @ v1.7.2
- extern/zenoh-kotlin (HEAD)
- extern/zenoh-demos (HEAD)
- extern/cargo-ndk @ v4.1.2

### 7. Create minimal C shim stubs
- src/zenoh_dart.h — FFI_PLUGIN_EXPORT macro, no functions, no #include <zenoh.h>
- src/zenoh_dart.c — just #include "zenoh_dart.h"
- src/CMakeLists.txt — minimal, no zenoh-c linking

### 8. Create minimal Dart files
- packages/zenoh/lib/zenoh.dart — empty barrel
- packages/zenoh/lib/src/native_lib.dart — DynamicLibrary loader

### 9. Create ffigen.yaml + generate bindings
At packages/zenoh/ffigen.yaml, then `fvm dart run ffigen`

### 10. Config files
.gitattributes (*.so binary), .gitignore, analysis_options.yaml

### 11. Copy docs from zenoh_dart + update Phase 0 and P1

### 12. Create CLAUDE.md

### 13. Verify
- fvm dart run melos bootstrap
- fvm dart analyze packages/zenoh
- fvm dart run ffigen (from packages/zenoh/)
- fvm dart test (from packages/zenoh/)
- git submodule status

### 14. Initial commit
"feat: scaffold zenoh-dart monorepo with pure Dart zenoh package"

## What the Scaffold Contains
- Monorepo with Dart workspace + Melos 7.4.0
- packages/zenoh/ — minimal FFI loader, no implementation
- src/zenoh_dart.{h,c} — trivial stubs
- 5 submodules at pinned versions
- All 20 phase specs (0 & P1 updated for pure Dart)

- NO prebuilts, NO Dart SDK headers (built/extracted during TDD)
